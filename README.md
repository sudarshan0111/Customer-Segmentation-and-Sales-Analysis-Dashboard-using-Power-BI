# Customer Segmentation & Sales Analysis Dashboard

## Overview

This project focuses on customer segmentation and sales analysis using Power BI, DAX, and SQL. Customers are categorized into different value groups based on their purchasing behavior, allowing businesses to identify high-value customers and understand sales patterns more effectively.

The dashboard combines customer segmentation with sales analytics to support strategic business decisions.

## Features

- Customer Value Segmentation
  - High Value
  - Medium Value
  - Low Value
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Total Customers KPI
- Average Order Value
- Profit Margin Analysis
- Customer Value Distribution
- Sales by Category
- Profit by Category
- Sales by Region
- Sales by Segment
- Top 10 Customers by Sales
- Sales by Sub-Category
- Interactive Region Filter

## Technologies Used

- Power BI
- DAX
- MySQL
- SQL
- Microsoft Excel

## Dataset

The project uses the Sample Superstore dataset to analyze customer purchasing behavior and sales performance.

## Customer Segmentation Logic

| Customer Value | Criteria |
|----------------|----------|
| High Value | Sales ≥ 1000 |
| Medium Value | Sales ≥ 500 and < 1000 |
| Low Value | Sales < 500 |

Customer segmentation is implemented using calculated columns created with DAX.

## Dashboard Components

- KPI Cards
- Donut Charts
- Bar Charts
- Customer Segmentation Analysis
- Sales Performance Visualizations
- Interactive Filters

## Key Insights

- High-value customers contribute significantly to overall revenue.
- Consumer segment generates the highest sales.
- Technology products lead in revenue generation.
- West region records the highest sales performance.
- Top customers account for a substantial portion of total business revenue.
